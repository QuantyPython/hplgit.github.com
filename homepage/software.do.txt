======= Software =======

<img src="fig/Fortran1.jpg" width=300 align="right">

 * "DocOnce": "http://hplgit.github.io/doconce/doc/web/index.html" for a unified approach to scientific publishing in various formats (LaTeX/PDF, Sphinx, Bootstrap HTML, IPython notebook, Markdown, wiki, reveal.js, deck.js, LaTeX Beamer, Remark, ...)
 * "Pysketcher": "https://github.com/hplgit/pysketcher" for specifying scientific drawings/sketches through programs
 * "Odespy": "http://hplgit.github.io/odespy/doc/web/index.html" for a unified Python interface to lots of software for solving ordinary differential equations
 * "Parampool": "http://hplgit.github.io/parampool/doc/web/index.html" for handling interfaces to simulation codes
 * "SciTools": "https://code.google.com/p/scitools/" for various scientific computing and plotting capabilities in Python, built on `numpy`, `scipy`, `ScientificPython`, `matplotlib`, `gnuplot`, ...
 * "Software examples": "http://hplgit.github.io/scipro-primer/"
   from the book *A Primer on Scientific Programming with Python*
 * "Bumpy": "http://hplgit.github.io/bumpy/doc/web/index.html" for introductory physics examples on scientific computing with Python
 * FEniCS tutorial: "PDF": "http://hplgit.github.io/INF5620/doc/pub/fenics_tutorial1.1-4print.pdf", "HTML": "http://hplgit.github.io/INF5620/doc/pub/fenics_tutorial1.1/index.html" for introducing the FEniCS software for solving partial differential equations by the finite element method
 * "FEniCS mixed": "http://hplgit.github.io/fenics-mixed/doc/web/index.html" for combining FEniCS with external third-party software

Former software activities:

 * "Diffpack": "http://diffpack.com": a general package for solving partial differential equations by the finite element method (work 1991-2003)
