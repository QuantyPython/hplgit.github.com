Running an application (old-style)

Run a stand-alone program: 

cmd = 'myprog -c file.1 -p -f -q > res'
failure = os.system(cmd)
if failure:
    print cmd, 'failed'
    sys.exit(1)

Redirect output from the application to a list of lines: 

pipe = os.popen(cmd)
output = pipe.readlines()
pipe.close()

for line in output:
  # process line
