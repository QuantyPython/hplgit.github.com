The purpose of this page is to exemplify how to write scientific reports various formats.

  • Plain HTML made from HTML source
  • HTML with LaTeX math made from MathJax-extended HTML source
  • PDF made from plain LaTeX source
  • PDF for electronic view made from LaTeX source generated from Doconce source (where URLs and references are hyperlinks)
  • PDF for printing made from LaTeX source generated from Doconce source (where linked URLs appear as footnotes)
  • PDF for viewing on phones made from LaTeX source generated from Doconce source
  • Sphinx (default layout) made from Sphinx-extended reStructuredText source generated from Doconce (other layouts: sphinx-ADCtheme, sphinx-agni, sphinx-agogo, sphinx-basic, sphinx-bootstrap, sphinx-cbc, sphinx-classy, sphinx-cloud, sphinx-default, sphinx-epub, sphinx-fenics, sphinx-fenics_minimal, sphinx-flask, sphinx-haiku, sphinx-impressjs, sphinx-jal, sphinx-nature, sphinx-pylons, sphinx-pyramid, sphinx-redcloud, sphinx-scipy_lectures, sphinx-scrolls, sphinx-slim-agogo, sphinx-sphinxdoc, sphinx-traditional, sphinx-vlinux-theme)
  • HTML with LaTeX math made from Doconce source
  • HTML with LaTeX math made from Doconce source with the "solarized" HTML style
  • HTML with LaTeX math made from Doconce source with the Vagrant template
  • HTML with LaTeX math made from Doconce source with the GitHub "minimal" theme template
  • HTML with LaTeX math made from Markdown generated from Doconce source
  • IPython notebook made from IPython notebook .ipynb file generated from Doconce source
  • MediaWiki made from MediaWiki format generated from Doconce source
