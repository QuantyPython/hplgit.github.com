Examples of scientific reports in different formats

Hans Petter Langtangen [1, 2]

[1] Simula
[2] Univ. of Oslo

Oct 3, 2014

The purpose of this page is to exemplify how to write scientific reports in various formats: LaTeX PDF, HTML, Sphinx, IPython notebook, HTML blog post, etc.

Formats generated from native sources

  • Plain HTML with primitive text-based math, made from HTML source
  • HTML with LaTeX math made from MathJax-extended HTML source
  • PDF made from plain LaTeX source

Bootstrap styles

DocOnce supports a range of HTML styles based on the Bootstrap framework. Such styles are seen all over the Internet and have a lot of functionality, e.g., responsive pages for smart phones and large screen. Here are some Bootstrap-based HTML styles: plain bootstrap made from DocOnce source, bloodish, FlatUT, bluegray. And here are some Bootswatch styles: cosmo, journal, readable, cyborg. .
