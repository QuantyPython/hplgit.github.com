The purpose of this page is to exemplify how to write scientific reports in the following formats:

  • Plain HTML made from HTML source
  • HTML with LaTeX math made from MathJax-extended HTML source
  • PDF made from LaTeX source generated from Doconce source
  • Sphinx (default layout) made from Sphinx-extended reStructuredText source generated from Doconce (other layouts: sphinx-agni, sphinx-agogo, sphinx-basic, sphinx-cbc, sphinx-classy, sphinx-cloud, sphinx-default, sphinx-epub, sphinx-fenics, sphinx-fenics_minimal, sphinx-flask, sphinx-haiku, sphinx-jal, sphinx-nature, sphinx-pylons, sphinx-pyramid, sphinx-redcloud, sphinx-scrolls, sphinx-slim-agogo, sphinx-solarized, sphinx-sphinxdoc, sphinx-traditional, sphinx-vlinux-theme)
  • HTML with LaTeX math made from Doconce source
