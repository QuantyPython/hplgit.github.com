class linear.Linear(a, b)[source]¶

Class for representing linear functions \(ax^2+b\).

>>> line = Linear(a=1, b=-2)
>>> line.value(4)
2.0
>>> line.root()
2.0
>>> print line
1.0*x - 2.0

class linear.Line(p1, p2)[source]¶

Compute the straight line that goes through two points p1 and p2.

Example:

>>> line = Line((1,0), (-4,1))
>>> print line
-0.2*x + 0.2
>>> line.value(1)
0.0
>>> line.value(-4)
1.0
